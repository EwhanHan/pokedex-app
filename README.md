# Pokedex

An index of Pokemon, built with Create React App.

# Developing

1. Install decencies

```
// yarn
yarn

// npm
npm install
```

2. Start dev server on localhost:3000

```
// yarn
yarn start

// npm
npm start
```

Optional: Run lint

```
// yarn
yarn run lint

// npm
npm run lint
```

Optional: Run lint

```
// yarn
yarn run format

// npm
npm run format
```
